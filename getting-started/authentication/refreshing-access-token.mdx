# `POST /auth/issue/refresh`

#### Request

<ParamField body="grant_type" type="string" required>
  The grant type of the authentication
</ParamField>

<ParamField body="refresh_token" type="string" required>
  A Base64 Encoded String used to refresh an expired access_token
</ParamField>


#### Response

<ResponseField name="businessId" type="string" required>
  The accountId of the merchant/business that is connecting to the Nomba system via API
</ResponseField>

<ResponseField name="access_token" type="string" required>
  A JWT (Json Web Token) token used to grant access to resources on the Nomba API
</ResponseField>

<ResponseField name="refresh_token" type="string" required>
  A Base64 Encoded String used to refresh an expired access_token
</ResponseField>

<ResponseField name="expiresAt" type="string" required>
  The date and time this entity was created. This value uses Coordinated Universal Time (UTC) and ISO 8601 format â€“ `YYYY-MM-DDThh:mm:ssZ`
</ResponseField>


<RequestExample>
```bash Request
curl --request POST \
  --url https://api.nomba.com/v1/auth/token/refresh \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --header 'accountId: <accountid>' \
  --data '{
  "grant_type": "refresh_token",
  "refresh_token": "01h4gdx2tctxfjgacbdwrcvs5d1688473602892"
}'
```
</RequestExample>
