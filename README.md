# Nomba Developer Documentation

The official developer documentation for the Nomba API, built with [Mintlify](https://mintlify.com).

---

## Running the Docs locally

Install the [Mintlify CLI](https://www.npmjs.com/package/mintlify):

```bash
npm i -g mintlify
```

Run the dev server from the root of the repo (where `docs.json` is):

```bash
mintlify dev
```

#### Troubleshooting

- **Mintlify dev isn't running** — Run `mintlify install` to re-install dependencies.
- **Page loads as a 404** — Make sure you are running from the folder that contains `docs.json`.

## AI Chat Backend (Optional)

The repo includes an AI-powered chat widget backed by a FastAPI server. It lets developers ask questions about the Nomba APIs directly from the documentation, with answers generated by Claude (Haiku) using the documentation as context.

### Prerequisites

- Python 3.10+
- An [Anthropic API key](https://console.anthropic.com/)

### Setup

**1. Navigate to the backend directory**

```bash
cd ai-chat-backend
```

**2. Create a virtual environment and install dependencies**

```bash
python -m venv venv
source venv/bin/activate      # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

**3. Configure your API key**

Copy the example env file and add your key:

```bash
cp .env.example .env
```

Open `.env` and set:

```
ANTHROPIC_API_KEY=your_api_key_here
```

**4. Start the server**

```bash
uvicorn main:app --reload
```

The backend will be available at `http://localhost:8000`. On startup it loads all documentation pages (including the OpenAPI spec) and any previously cached responses.


### How it works

| Component | Detail |
|---|---|
| Model | Claude Haiku (fast, low cost) |
| Context | Top 3 most relevant doc pages retrieved per question via keyword search |
