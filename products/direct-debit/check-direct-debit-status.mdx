# `GET /v1/direct-debits/status?mandateId=<mandateId>`


#### Response body

<ResponseField name="code" type="string" required>
  Response code
</ResponseField>

<ResponseField name="description" type="string" required>
  Response description
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="object"  defaultOpen="true">
    <ResponseField name="mandateId" type="string" required>
      Mandate ID
    </ResponseField>
    <ResponseField name="customerAccountName" type="string" required>
      Customer Account Name
    </ResponseField>
    <ResponseField name="customerAccountNumber" type="string" required>
      Customer Account Number
    </ResponseField>
    <ResponseField name="mandateStatus" type="string" required>
      Mandate Status
    </ResponseField>
    <ResponseField name="rejectionReason" type="string" required>
      Rejection Reason
    </ResponseField>
    <ResponseField name="mandateAdviceStatus" type="string" required>
      Mandate Advice Status
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="message" type="string" required>
  Message
</ResponseField>

<ResponseField name="status" type="boolean" required>
  status
</ResponseField>


<RequestExample>
```bash Request
curl --location 'http://api.nomba.com/v1/direct-debits/status?mandateId=<mandateId>' \
--header 'Authorization: Bearer [token]' \
--header 'accountId: <accountId>' \
--header 'Content-Type: application/json' \
--data ''
```
</RequestExample>
