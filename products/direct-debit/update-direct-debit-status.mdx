# `PUT /v1/direct-debits/update-status`

#### Request body

<ParamField body="mandateId" type="string" required>
  This is the mandate ID.
</ParamField>

<ParamField body="status" type="string" required>
  The status you want to change to.
</ParamField>

#### Response body

<ResponseField name="code" type="string" required>
  Response code
</ResponseField>

<ResponseField name="description" type="string" required>
  Response description
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="object"  defaultOpen="true">
    <ResponseField name="mandateId" type="string" required>
      Mandate ID
    </ResponseField>
    <ResponseField name="mandateStatus" type="string" required>
      Mandate Status
    </ResponseField>
    <ResponseField name="mandateAdviceStatus" type="string" required>
      Mandate Advice Status
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="message" type="string" required>
  Message
</ResponseField>

<ResponseField name="status" type="boolean" required>
  status
</ResponseField>


<RequestExample>
```bash Request
curl --location 'https://api.nomba.com/v1/direct-debits' \
--header 'Authorization: Bearer [token]' \
--header 'accountId: <accountId>' \
--header 'Content-Type: application/json' \
--data-raw '{
  "mandateId": "63fba2a159bef38d97f462d9b473306bed944544",
  "status": "SUSPEND",
}'
```
</RequestExample>
