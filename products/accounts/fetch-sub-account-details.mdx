# `GET /v1/accounts`

#### Query parameters

<ParamField query="accountRef" type="string" optional>
  The unique reference you provided when creating the account
</ParamField>

<ParamField query="accountId" type="string" optional>
  The accountId you received as part of the response when the account was created
</ParamField>

#### Response body

<ResponseField name="code" type="string" required>
  Response code
</ResponseField>

<ResponseField name="description" type="string" required>
  Response description
</ResponseField>

<ResponseField name="data" type="object" required>
  <Expandable title="object"  defaultOpen="true">
    <ResponseField name="cursor" type="string">
      Cursor for pagination. It will be empty if there is no more page to scroll to.
    </ResponseField>
    <ResponseField name="results" type="object[]">
        <Expandable title="object"  defaultOpen="true">
            <ResponseField name="createdAt" type="string" required>
                Creation timestamp.
            </ResponseField>
            <ResponseField name="accountId" type="string" required>
                Account ID.
            </ResponseField>
            <ResponseField name="accountHolderId" type="string" required>
                Account holder ID.
            </ResponseField>
            <ResponseField name="accountRef" type="string" required>
                Account reference.
            </ResponseField>
            <ResponseField name="phoneNumber" type="string" required>
                Phone number.
            </ResponseField>
            <ResponseField name="email" type="string" required>
                Email address.
            </ResponseField>
            <ResponseField name="bvn" type="string" required>
                Bank Verification Number (BVN).
            </ResponseField>
            <ResponseField name="status" type="enum<string>" required>
                Account status.
            </ResponseField>
            <ResponseField name="type" type="enum<string>" required>
                Account type.
            </ResponseField>
            <ResponseField name="accountName" type="string" required>
                Account holder's name.
            </ResponseField>
            <ResponseField name="currency" type="string" required>
                Currency code.
            </ResponseField>
            <ResponseField name="callbackUrl" type="string">
                Callback url.
            </ResponseField>
            <ResponseField name="expiryDate" type="string">
                Expiry date.
            </ResponseField>
        </Expandable>
    </ResponseField>
    
  </Expandable>
</ResponseField>

<RequestExample>
```bash Request
curl --request GET \
  --url https://api.nomba.com/v1/accounts \
  --header 'Authorization: Bearer <token>' \
  --header 'accountId: <accountid>'
```
</RequestExample>
