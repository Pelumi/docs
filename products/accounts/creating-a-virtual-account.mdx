# `POST /v1/accounts/virtual`

#### Request body

<ParamField body="accountRef" type="string" required>
  Account reference
</ParamField>

<ParamField body="accountName" type="string" required>
  Account holder's name
</ParamField>

<ParamField body="currency" type="string" required>
  Currency code
</ParamField>

<ParamField body="expiryDate" type="string" optional>
  Expiry Date 
  (This field allows you to customize the expiry period for this account. The default time however is 5 minutes)
</ParamField>

#### Response body

<ResponseField name="code" type="string" required>
  Response code
</ResponseField>

<ResponseField name="description" type="string" required>
  Response description
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="object"  defaultOpen="true">
    <ResponseField name="createdAt" type="string" required>
      Creation timestamp.
    </ResponseField>
    <ResponseField name="accountId" type="string" required>
      Account ID.
    </ResponseField>
    <ResponseField name="accountHolderId" type="string" required>
      Account holder ID.
    </ResponseField>
    <ResponseField name="accountRef" type="string" required>
      Account reference.
    </ResponseField>
    <ResponseField name="bvn" type="string" required>
      Bank Verification Number (BVN).
    </ResponseField>
    <ResponseField name="status" type="enum<string>" required>
      Account status.
    </ResponseField>
    <ResponseField name="type" type="enum<string>" required>
      Account type.
    </ResponseField>
    <ResponseField name="accountName" type="string" required>
      Account holder's name.
    </ResponseField>
    <ResponseField name="banks" type="object[]" required>
      List of associated bank accounts.
      <Expandable title="object"  defaultOpen="true">
        <ResponseField name="bankAccountNumber" type="string" required>
            Bank account number.
        </ResponseField>
        <ResponseField name="bankName" type="string" required>
            Bank name.
        </ResponseField>
        <ResponseField name="bankAccountName" type="string" required>
            Bank account holder name.
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="currency" type="string" required>
      Currency code.
    </ResponseField>
    <ResponseField name="expiryDate" type="string" optional>
      Expiry date of the account created.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash Request
curl --request POST \
  --url https://api.nomba.com/v1/accounts/virtual \
  --header 'Authorization: Bearer <token>' \
  --header 'Content-Type: application/json' \
  --header 'accountId: <parent accountId>' \
  --data '{
  "accountRef": "1oWbJQQHLyQqqf1SwxjSpudeA21",
  "accountName": "Daniel Scorsese",
  "currency": "NGN",
  "expiryDate": "2024-06-17 04:55:00"
}'
```
</RequestExample>
