---
title: 'Fetch Parent Account Details'
description: 'Retrieve details of a parent account'
---

### `GET /v1/accounts/fetch-parent-account`

Use this endpoint to fetch details of a parent account by providing the account ID in the request header.

#### Response body

<ResponseField name="status" type="boolean" required>
  Indicates whether the request was successful.
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="Parent account details" defaultOpen="true">
    <ResponseField name="bankAccountNumber" type="string" required>
      Bank account number.
    </ResponseField>
    <ResponseField name="bankAccountName" type="string" required>
      Bank account name.
    </ResponseField>
    <ResponseField name="bankName" type="string" required>
      Name of the bank linked to the account.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash Request
curl --request GET \
  --url https://api.nomba.com/v1/accounts/fetch-parent-account \
  --header 'Authorization: Bearer <token>' \
  --header 'accountId: <accountid>'
```
</RequestExample>

<ResponseExample>
```json Success response
{
    "data": {
        "bankAccountNumber": "21061XXXXXX",
        "bankAccountName": "Nomba/Praise God Ltd",
        "bankName": "Nombank(Amucha) MFB"
    },
    "status": true
}
```
</ResponseExample>
