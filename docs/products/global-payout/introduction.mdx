GlobalPayout enables cross-border fund disbursements, giving you full control over international transfers from initiation to completion. Whether you're sending funds via SWIFT, converting currencies between accounts, or tracking a payment in real time, GlobalPayout provides the tools to manage every step of the process.

## How it works

<Steps titleSize="p">
  <Step title="Fetch the latest exchange rate">
    Before initiating a cross-currency transfer, call the exchange rates endpoint to retrieve accurate, up-to-date conversion values. This ensures your customers see the correct rate before confirming a transaction.
  </Step>
  <Step title="Convert money">
    Call the ConvertMoney endpoint to calculate the exact destination amount after conversion. This returns a locked exchange rate ID that you will reference in the next step.
  </Step>
  <Step title="Authorize the transfer">
    Submit the full transfer details — recipient account, payment method, currencies, and auth code — to authorize and initiate the disbursement.
  </Step>
  <Step title="Track the transaction">
    Use the FetchTransaction endpoint with the returned `wtTransactionId` to poll for the current status of the transfer at any point in its lifecycle.
  </Step>
</Steps>
