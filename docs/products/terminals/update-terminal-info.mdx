# `PUT /v1/terminals/bulk/update`

*buck update - need clarity*

<CodeGroup>
```bash Request
curl --location --request PUT ' https://api.nomba.com/v1/terminals/bulk/update' \
--header 'Authorization: Bearer <token>' \
--header 'accountId: <accountId>' \
--header 'Content-Type: application/json' \
--data '{
    "data": [
        {
            "merchantName": "Testing",
            "terminalId": "2KUDXXXXXX",
            "terminalLabel": "Hello",
            "narrationName": "Hello"
        }
    ]
}'
```

```json Response
{
  "code": "00",
  "description": "SUCCESS"
}
```
</CodeGroup>
