You can effortlessly create your API keys within your Nomba dashboard. Navigate to your settings and click on the Webhook & API Keys tab. You will find a button exactly like this. Click on it to generate your keys.

<Frame caption="Generate API Key">
    <img src="/images/generate-keys.png" style={{ borderRadius: '0.5rem' }} loading="lazy" />
</Frame>

Kindly follow the steps illustrated in the above image to generate your API Key. Once the API Keys are generated, you can copy them for integration with the Nomba API. After copying, make an effort to [Obtain a token.](/getting-started/authentication/obtaining-access-token)

<Frame caption="Retrieve your API Keys">
    <img src="/images/api-keys-page.png" style={{ borderRadius: '0.5rem' }} loading="lazy" />
</Frame>

<Warning>
  Ensure you protect your secret keys at all times, as they can be used to gain access to your account.  
</Warning>
