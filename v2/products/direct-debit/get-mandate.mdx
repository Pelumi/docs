# `GET /v1/direct-debits/<mandateId>`


#### Response body

<ResponseField name="code" type="string" required>
  Response code
</ResponseField>

<ResponseField name="description" type="string" required>
  Response description
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="object"  defaultOpen="true">
    <ResponseField name="mandateId" type="string" required>
      Mandate ID
    </ResponseField>
    <ResponseField name="customerAccountName" type="string" required>
      Customer Account Name
    </ResponseField>
    <ResponseField name="customerAccountNumber" type="string" required>
      Customer Account Number
    </ResponseField>
      <ResponseField name="customerPhoneNumber" type="string" required>
          Customer Phone Number
      </ResponseField>
      <ResponseField name="customerEmail" type="string" required>
          Customer Email
      </ResponseField>
      <ResponseField name="customerName" type="string" required>
          Customer Name
      </ResponseField>
      <ResponseField name="customerAddress" type="string" required>
          Customer Address
      </ResponseField>
    <ResponseField name="status" type="string" required>
      Mandate Status
    </ResponseField>
    <ResponseField name="bankCode" type="string" required>
      Bank Code
    </ResponseField>
    <ResponseField name="amount" type="string" required>
      Mandate Amount
    </ResponseField>
      <ResponseField name="frequency" type="string" required>
          Mandate Frequency
      </ResponseField>
      <ResponseField name="merchantReference" type="string" required>
          Mandate Merchant Reference
      </ResponseField>
    <ResponseField name="mandateAdviceStatus" type="string" required>
      Mandate Advice Status
    </ResponseField>
    <ResponseField name="startDate" type="string" required>
      Mandate Start Date
    </ResponseField>
    <ResponseField name="endDate" type="string" required>
      Mandate End Date
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="message" type="string" required>
  Message
</ResponseField>

<ResponseField name="status" type="boolean" required>
  status
</ResponseField>


<CodeGroup>
```bash Request
curl --location 'http://api.nomba.com/v1/direct-debits/<mandateId>' \
--header 'Authorization: Bearer [token]' \
--header 'accountId: <accountId>' \
--header 'Content-Type: application/json' \
--data ''
```
