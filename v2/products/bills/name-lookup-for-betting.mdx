# `GET /v1/bill/betting/lookup`

#### Query parameters

<ParamField query="providerId" type="string">
    The code name of the betting provider as gotten when the list of betting providers is fetched e.g. bet9ja
</ParamField>

<ParamField query="customerId" type="string">
    The unique identifier of the customer on the provider's platform
</ParamField>

#### Response body

<ResponseField name="code" type="string" required>
    Response code
</ResponseField>

<ResponseField name="description" type="string" required>
    Response description
</ResponseField>

<ResponseField name="data" type="string" required>
    Data containing customer's name if present
</ResponseField>

<ResponseField name="message" type="string" required>
    Response message
</ResponseField>

<RequestExample>
    ```bash Request
    curl --location GET \
    --url https://api.nomba.com/v1/bill/betting/lookup \
    --header 'accountId: <accountId>' \
    --header 'Authorization: <token>'
        ```
</RequestExample>

<ResponseExample>
    ```json Success response
    {
        "code": "00",
        "description": "SUCCESS",
        "data": "John Doe",
        "message": "SUCCESS"
    }
    ```
</ResponseExample>
